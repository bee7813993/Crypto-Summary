初回セットアップ

API キーを Web 画面から登録するために必要な
暗号化マスター鍵を設定します。

Google Cloud Console で発行した OAuth クライアントの値を入力してください。リダイレクトURI には「BASE_URL + /auth/callback」を登録します。

このキーで取引所・ウォレットの API キーを暗号化します。安全な場所にバックアップしてください。

指定したアカウントだけが、Web からシステム設定（Etherscan/Helius キー）を編集できます。未設定でも後から .env で変更できます。

スキップすると、Web 画面から API キーを登録する機能が使えません。

管理者設定

Google OAuth（ログイン設定）

マルチユーザーモードでのログインに使用する Google OAuth 設定です。リダイレクト URI には「BASE_URL + /auth/callback」を登録してください。

暗号化マスター鍵（CS_SECRET_KEY）

CS_SECRET_KEY は初回セットアップ時にのみ Web から設定できます。以降の変更はサーバーの設定ファイルか環境変数で行ってください。

管理者メールアドレス（ADMIN_EMAILS）

この管理者設定画面へのアクセス権を持つ Google アカウントのメールアドレスです。カンマ区切りで複数指定できます。

スキャン用 API キー

全ユーザー共通のスキャン用 API キーです。ウォレットアドレスの同期に使用します。環境変数より優先されます。

CoinGecko API Key

価格・推移グラフの取得に使用します。無料の Demo プランで取得できます。設定するとレート制限が緩和されます（任意）。
